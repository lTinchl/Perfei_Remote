# Perfei_Remote
## 简介

此项目根据B站开源大佬Fei_行家所发布的Remote_V3.0代码的基础移植了u8g2并优化了nrf发包与摇杆状态采集，与无人机锁定状态和信号检测(可惜原理图没接硬件I2C,不然可以写高帧率动画了)。

## 如何使用
1. 通过Code下载压缩包解压使用
2. 使用git clone部署至本地。
