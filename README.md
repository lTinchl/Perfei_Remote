# Perfei_Remote
## 简介

此项目由B站开源大佬Fei_行家所发布的Remote3.0代码以此基础移植了u8g2并优化了nrf发包与摇杆状态采集，与无人机锁定状态和信号检测。

## 如何使用
在Code下载压缩包解压使用或着使用git clone部署至本地。

## 问题和建议
